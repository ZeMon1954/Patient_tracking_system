# Skill: known-bugs

## Trigger
เรียกใช้ Skill นี้โดยอัตโนมัติเมื่อ:
- ผู้ใช้แจ้งว่าเจอ Bug, Error, หรือปัญหาที่โปรแกรมทำงานไม่ถูกต้อง
- กำลังวิเคราะห์หาสาเหตุของปัญหา (Troubleshooting)

## Context
ไฟล์นี้คือ **"ฐานข้อมูลความรู้ (Knowledge Base)"** ที่รวบรวมประวัติบั๊กที่เคยเจอในโปรเจกต์นี้และวิธีแก้ไขที่เคยใช้สำเร็จมาแล้ว เพื่อให้ Claude สามารถเรียนรู้จากอดีตและแก้ปัญหาเดิมได้ทันทีโดยไม่ต้องลองผิดลองถูกใหม่

เมื่อเจอปัญหาใหม่ ให้ตรวจสอบในรายการด้านล่างนี้ก่อนว่าตรงกับอาการที่เคยเจอหรือไม่
